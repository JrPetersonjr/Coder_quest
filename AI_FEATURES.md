## **üîß For Developers**

### **Using AI in Custom Content**

```javascript
// Example: Custom event with AI
async function onPlayerLevelUp() {
  const celebration = await AIConfig.generateDMNarrative(
    `The player has reached level ${player.level}. Generate a celebratory message.`
  );
  
  gameEngine.output(celebration, "highlight");
}

// Example: Procedural quest rewards
async function generateQuestReward() {
  const itemDescription = await AIConfig.generateContent(
    "item", 
    `rarity: rare, theme: cyberpunk`
  );
  
  return {
    name: "Ancient Artifact",
    description: itemDescription,
  };
}

// Example: Dynamic challenge
async function createChallenge(difficulty) {
  const challenge = await AIConfig.generateContent(
    "challenge",
    `difficulty: ${difficulty}, skill: coding`
  );
  
  return {
    prompt: challenge,
    type: "code-puzzle",
  };
}
```

## **üöÄ Performance Tips**

### **For Site Admins**

**Best for performance (fastest):**
1. Local LM Studio model (no network latency)
2. HuggingFace Pro (optimized servers)
3. HuggingFace Free (rate-limited but free)
4. Fallback content (instant, always works)

**Recommended setup:**
```javascript

AIConfig.setLocalModelURL('http://localhost:1234/v1/chat/completions');

// Production  
AIConfig.setAPIKey('huggingface', 'hf_YOUR_PRO_TOKEN');

// Or highest quality
AIConfig.setAPIKey('openai', 'sk_YOUR_KEY');
```

## **‚ùì FAQ**

### **Q: Will the game work if AI is disabled?**
A: Yes! The game works perfectly. Fallback content is used instead.

### **Q: Why is the oracle sometimes repetitive?**
A: If using fallback (no AI), it cycles through pre-written prophecies. Enable an AI provider for infinite variety.

### **Q: Can I use my own API key?**
A: Yes! Site admins can set API keys during deployment. See AI_INTEGRATION_GUIDE.md.

### **Q: Is AI generation slow?**
A: No! Most responses are <2 seconds. Local models are <500ms.

### **Q: What data is sent to AI services?**
A: Only the prompt text. No personal data. Check the service's privacy policy.

### **Q: Can I disable specific AI features?**
A: Yes! Use: `AIConfig.setFeature('crystalBall', false)` etc.
